# Tugas Besar Mata Kuliah Python 
## Analisis dan Prediksi Risiko Serangan Jantung

Tugas ini merupakan Tugas Besar mata kuliah Python yang bertujuan untuk melakukan analisis, visualisasi data, serta penerapan model machine learning sederhana menggunakan bahasa pemrograman Python.

Dataset yang digunakan adalah dataset kesehatan pasien yang berisi beberapa indikator medis untuk memprediksi risiko serangan jantung.

## Tujuan Proyek
1. Melakukan analisis dataset menggunakan Python  
2. Menampilkan visualisasi data untuk mendapatkan wawasan dari dataset  
3. Membangun model machine learning sederhana (Logistic Regression)  
4. Melakukan prediksi risiko serangan jantung berdasarkan data pasien  

---

## Dataset
Dataset yang digunakan adalah dataset kesehatan pasien yang berisi beberapa atribut,
antara lain:
- Usia
- Tekanan darah
- Kolesterol
- Detak jantung maksimum
- Risiko serangan jantung (output)

File dataset:
- `heartAttack.csv`

---

## Fitur Program
Program menyediakan beberapa menu utama, yaitu:
1. Menampilkan dataset beserta keterangan kolom
2. Menampilkan visualisasi data
3. Melatih model machine learning
4. Melakukan prediksi risiko serangan jantung berdasarkan data pasien baru
